# Contributing to Parking Management System

Thank you for your interest in contributing! This document provides guidelines and instructions for contributing to this project.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/parking-management-system-using-CV.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes
6. Commit: `git commit -m "Add your feature"`
7. Push: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Code Style

- Follow PEP 8 Python style guide
- Use meaningful variable and function names
- Add docstrings to functions and classes
- Keep functions focused and modular

## Testing

Before submitting a PR, please:
- Test your changes with sample videos
- Ensure no existing functionality is broken
- Update documentation if needed

## Pull Request Process

1. Update README.md if needed
2. Ensure your code follows the project's style
3. Add comments for complex logic
4. Update documentation for new features

## Questions?

Feel free to open an issue for questions or suggestions!
